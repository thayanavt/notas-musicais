# notas-musicais

Notas musicais é um CLI para ajudar na formação de escalas, acordes e campos harmônicos. Projeto desenvolvido pelo Eduardo Mendes e ministrado no curso "Construindo um pacote Python do zero",
disponível em https://www.youtube.com/watch?v=R3hCkU4EXgY&list=PLOQgLBuj2-3LiHhK1upnjpHiFzcJ472QS.
